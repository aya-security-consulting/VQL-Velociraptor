A curated repository of **Velociraptor Query Language (VQL)** content dedicated to **threat hunting, DFIR, detection engineering, and forensic investigations**.

## Overview

This repository centralizes a wide range of **VQL queries** designed to support security teams in identifying suspicious activity, accelerating investigations, and improving visibility across endpoints and environments.

The objective is to provide practical and reusable hunting content that can be leveraged during:

- Threat hunting activities
- Incident response engagements
- Digital forensic investigations
- Detection validation
- Compromise assessment
- Endpoint triage and collection

## What You Will Find

This repository lists and organizes VQL queries covering multiple use cases, including:

- **Persistence hunting**
- **Credential access detection**
- **Lateral movement investigation**
- **Execution and process analysis**
- **Windows event log collection and review**
- **Registry and scheduled task analysis**
- **Browser artifact hunting**
- **File system and suspicious file discovery**
- **Network-related artifacts**
- **User activity and timeline reconstruction**
- **Malware triage and forensic collection**
- **Threat actor tradecraft detection**

## Purpose

The goal of this project is to make VQL hunting content easier to access, review, and reuse for analysts, hunters, and responders.  
Each query is intended to help detect anomalies, uncover attacker behavior, or collect relevant forensic evidence in a fast and structured way.

## Target Audience

This repository is intended for:

- SOC analysts
- Threat hunters
- DFIR analysts
- Incident responders
- Detection engineers
- Blue team practitioners

Queries from this repository can be adapted depending on the investigation scope, the operating system, the artifact availability, and the maturity of the environment.

Before running any query in production, ensure it has been reviewed and validated against your operational constraints.
